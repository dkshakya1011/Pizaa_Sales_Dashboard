# Pizaa_Sales_Dashboard
## project objective
A SQL and Power BI project analyzing pizza sales data to uncover key business insights. Includes KPI calculations, sales trends, and category/size performance visualized through an interactive dashboard.

## Dataset used
- <a href = "https://github.com/dkshakya1011/Pizaa_Sales_Dashboard/blob/main/pizza_sales_1.xlsx">Dataset</a>

## Questions (KPIs)
- What is the Total Revenue?
- What is the Total Number of Orders?
- What is the Total Number of Pizzas Sold?
- What is the Average Order Value (AOV)?
- What is the Average Pizzas per Order?
